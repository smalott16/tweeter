# Tweeter Project

Tweeter is a simple, single-page Twitter clone. The application uses a back-end express server and several front-end tools including HTML, CSS, Node, JS, jQuery and AJAX.

## Key Features

- Users can enter "tweets" of 140 characters or less
- Error handling is in place to catch any tweets with too much content or no content at all.
- New tweets are added to the feed via AJAX requests 
- App is useable on both desktop and mobile screen sizes

## Getting Started

1. Fork this repository, then clone your fork of this repository.
2. Install dependencies using the `npm install` command.
3. Start the web server using the `npm run local` command. The app will be served at <http://localhost:8080/>.
4. Go to <http://localhost:8080/> in your browser.

## Dependencies

- Express
- Node 5.10.x or above
- body parser
- md5
- chance
